Automated User Story Generation with Large Language Models (2026)

Master’s Thesis Project • NLP • Prompt Engineering • LLM Evaluation

Proposed an LLM-based approach to automatically generate user stories from software requirements, addressing limitations of traditional ML/NLP methods in capturing “who”, “what”, and “why”.

  • Compared LLaMA and Mistral for requirement-to-user-story generation.
  • Tested prompt engineering strategies: few-shot, meta-prompting, and directional stimulus.
  • Evaluated quality with BLEU, ROUGE, METEOR, BERTScore + qualitative error analysis.
  • Findings: LLMs produce coherent and semantically correct user stories; few-shot improves results; LLaMA slightly stronger on complex/sequential cases.
